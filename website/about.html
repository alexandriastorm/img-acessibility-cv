We Care About Acessibility

We built this website to address the issue of accessibility within Search Engines.

We utilized the Azure Cognitive Services Computer Vision platform and built our own NLP model on top of that in order to create smart captioning for images for those who are visually impaired. This allows for users with disabilities to both see an auto-generated caption for the image they are hovering over in Image Search, and additinally have the description be read to them.

We are three explore interns in Microsoft AI + Research, specifcially under Answers Intelligence and Natural Language Understanding.

Our Project

High Quality Image Captioning

We analyze the image in general, including identifiying objects, landscapes and people with specific confidence levels. This is done utilizing Azure Cognitive Services.

Additional NLP Modeling

We added our own additions to our Natural Language Processing model, supporting accurate but more generic descriptions of the image if we did not reach 60% confidence. This increased accuracy of image detection 40% from the Azure Vision API.

Extending Search Engine Capabilities

This feature is a proposed accessibility feature for Bing Search. If a user has a visual disability and has accesibiliy features are turned on, they can hover over an image and get the general description of the photo read outloud to them.
